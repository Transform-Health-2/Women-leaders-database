Screen 6 — Nominations: Invited · notes

  • Source: Each row originates from a "I am nominating someone else" submission — only a profile link is captured
  • Awaiting: Admin sent invite, no response yet · Submitted: Nominee completed their own form → now in Pending queue
  • No response: Invite sent 7+ days ago with no action — auto-flag or manual resend
  • Decline: Admin chooses not to pursue this nomination → moves to Declined tab with optional note

Screen 7 — Nominations: Declined · notes

  • Declined by admin (nomination not suitable) or by nominee (opted out after receiving invite)

Admin UI — Route Map
/login → Screen 1 — Password gate
/dashboard → Screen 2 — Overview · counts from all Sheets tabs
/pending → Screen 3 — Self-submission queue · approve / reject
/pending/:id → Screen 4 — Submission detail · full profile preview
/live → Screen 5 — Live profiles · featured toggle
/nominations/invited → Screen 6 — Invited nominations · resend / decline
/nominations/declined → Screen 7 — Declined nominations · reinstate
/rejected → Rejected submissions (same layout as /live)
Google Sheets DB — Tab Structure
Pending
New self-submissions from the bespoke form. Each row awaits admin approve / reject.
Live
Approved profiles. Served via Sheets API to the public directory and the Outreach Dashboard.
Rejected
Declined submissions. Audit trail only. Not served publicly. Admin can reinstate.
Nominations
Inbound "nominate someone" submissions. Stores profile link + nominator email. Admin manages invite status here.
Outreach
Internal-only tab. Tracks outreach activity, invite history, follow-up status. Feeds the Outreach Dashboard view.
